🧮 Calculadora Vibe

Calculadora web moderna desarrollada con HTML, CSS y JavaScript puro,
con sistema de historial dinámico y modo claro/oscuro.

Proyecto enfocado en práctica de lógica, manipulación del DOM y
organización de código frontend sin frameworks.

------------------------------------------------------------------------

🚀 Demo

👉 https://msanti14.github.io/vibe-calculator/

------------------------------------------------------------------------

✨ Features

-   Operaciones matemáticas básicas
-   Historial dinámico con scroll interno
-   Toggle de tema (Dark / Light Mode)
-   Interfaz responsive
-   Código modular y organizado
-   Sin dependencias externas

------------------------------------------------------------------------

🛠️ Tecnologías utilizadas

-   HTML5
-   CSS3 (Flexbox + variables CSS)
-   JavaScript Vanilla (ES6)

------------------------------------------------------------------------

📂 Estructura del proyecto

calculadora-vibe/
│
├── index.html

└── README.md

------------------------------------------------------------------------

🎯 Objetivos del proyecto

-   Practicar manipulación del DOM
-   Implementar estado visual dinámico (theme switcher)
-   Controlar renderizado de listas con overflow
-   Mejorar organización de lógica en JS
-   Preparar proyecto apto para portfolio profesional

------------------------------------------------------------------------

🧠 Aprendizajes clave

-   Manejo de eventos
-   Control de estado (tema activo)
-   Renderizado dinámico del historial
-   Uso correcto de overflow-y para scroll interno
-   Separación de responsabilidades (HTML / CSS / JS)

------------------------------------------------------------------------

🌗 Modo Claro / Oscuro

El botón toggle permite cambiar el tema visual de la calculadora y el
historial utilizando variables CSS, manteniendo una arquitectura limpia
y escalable.

------------------------------------------------------------------------

📦 Cómo usarlo localmente

1.  Clonar el repositorio:

git clone https://github.com/msanti14/vibe-calculator

2.  Abrir index.html en el navegador.

------------------------------------------------------------------------

🌍 Deploy en GitHub Pages

1.  Ir a Settings
2.  Sección Pages
3.  En “Source” seleccionar la rama main
4.  Guardar

GitHub generará automáticamente la URL pública.
