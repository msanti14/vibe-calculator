# Vibe Calculator v1

Mi primera aplicación web — una calculadora con historial y toggle de tema.

Construida con HTML, CSS y JavaScript vanilla, sin frameworks ni herramientas de build.

## Features

- Operaciones básicas: suma, resta, multiplicación, división
- Historial de operaciones con persistencia en localStorage
- Click en item del historial para restaurar la expresión
- Dark / light mode con persistencia
- Soporte de teclado completo

## Stack

- HTML + CSS + JavaScript vanilla
- Sin dependencias, sin build tool
